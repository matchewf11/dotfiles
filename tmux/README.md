# TMUX Cheat Sheet

## Prefix Key

```
Ctrl + b
```

> All tmux commands start with this by default.

## Sessions

| Command                     | Description                              |
| --------------------------- | ---------------------------------------- |
| `tmux new -s name`          | Create a new session named `name`        |
| `tmux ls`                   | List all sessions                        |
| `tmux attach -t name`       | Attach to a session                      |
| `tmux detach`               | Detach from current session (`Ctrl+b d`) |
| `tmux kill-session -t name` | Kill a session                           |
| `tmux kill-server`          | Kill all sessions                        |

## Windows (Tabs)

| Shortcut   | Description          |
| ---------- | -------------------- |
| `Ctrl+b c` | Create new window    |
| `Ctrl+b n` | Next window          |
| `Ctrl+b p` | Previous window      |
| `Ctrl+b ,` | Rename window        |
| `Ctrl+b &` | Close current window |
| `Ctrl+b w` | List windows         |
| `Ctrl+b #` | Goto window #        |

## Panes (Splits)

| Shortcut           | Description         |
| ------------------ | ------------------- |
| `Ctrl+b %`         | Split vertically    |
| `Ctrl+b "`         | Split horizontally  |
| `Ctrl+b o`         | Switch to next pane |
| `Ctrl+b ;`         | Switch to last pane |
| `Ctrl+b x`         | Close current pane  |
| `Ctrl+b z`         | Toggle pane zoom    |
| `Ctrl+b arrow`     | Move between panes  |
| `Ctrl+b Alt+arrow` | Resize panes        |

## Navigation

| Shortcut   | Description              |
| ---------- | ------------------------ |
| `Ctrl+b [` | Enter copy mode (scroll) |
| `Ctrl+b ]` | Paste buffer             |
| `Ctrl+b t` | Show clock               |
| `Ctrl+b q` | Show pane numbers        |

## Other Useful Commands

| Command                         | Description                      |
| ------------------------------- | -------------------------------- |
| `tmux source-file ~/.tmux.conf` | Reload config                    |
| `set -g mouse on`               | Enable mouse support (in config) |
| `Ctrl+b ?`                      | Show all shortcuts               |

## Tips

* Detach and reattach sessions to keep long-running tasks alive.
* Use panes + windows to organize projects.
* Enable mouse support to click and select windows/panes easily.
* Consider changing prefix to `Ctrl+a` if you come from GNU Screen:

```bash
unbind C-b
set -g prefix C-a
bind C-a send-prefix
```
